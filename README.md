# 🏧 ATM Bank Simulator

A terminal-based ATM banking simulation written in Python.
Built for educational purposes.

---

## 📋 Requirements

Before running, install the required library:
```
pip install pystyle
```

If you get an `externally-managed-environment` error:
```
pip install pystyle --break-system-packages
```

---

## ▶️ How to Run
```
python "bank simulator python.py"
```

---

## ⚙️ VS Code Users

If the module is not found, select the correct interpreter:

`Ctrl+Shift+P` → `Python: Select Interpreter` → choose `pythoncore-3.14-64`
